Projeto frontend chamado DevQuiz. Aplicacao single-page em React criada com Vite para executar um quiz de multipla escolha carregando perguntas de um arquivo JSON local em public/data/questions.json. Estrutura principal: src/App.jsx renderiza src/components/Quiz.jsx; componentes auxiliares em src/components (QuestionCard, ResultScreen, ProgressBar); estilos globais em src/index.css; configuracao base em package.json, vite.config.js e index.html.
